# ai-confidence

> Confidence visualization | 置信度可视化

## Basic Usage

```html
<ai-confidence value="0.85"></ai-confidence>
<ai-confidence value="0.6" showLabel></ai-confidence>
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| value | `number` | `0` | Confidence value 0-1 |
| showLabel | `boolean` | `false` | Show percentage label |
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | Size variant |
