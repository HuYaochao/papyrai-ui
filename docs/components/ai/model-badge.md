# ai-model-badge

> Model badge with auto-coloring | 自动配色的模型徽章

## Basic Usage

```html
<ai-model-badge model="gpt-4"></ai-model-badge>
<ai-model-badge model="claude-3" size="sm"></ai-model-badge>
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| model | `string` | `''` | Model name |
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | Badge size |
| showIcon | `boolean` | `true` | Show model icon |
