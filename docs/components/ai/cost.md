# ai-cost

> Token cost display | Token 消耗显示

## Basic Usage

```html
<ai-cost inputTokens="500" outputTokens="1000"></ai-cost>
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| inputTokens | `number` | `0` | Input token count |
| outputTokens | `number` | `0` | Output token count |
| showDetail | `boolean` | `false` | Show detailed breakdown |
