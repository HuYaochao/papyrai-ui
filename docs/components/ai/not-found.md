# ai-not-found

> "AI thinks it exists but doesn't" | AI 以为存在但不存在

## Basic Usage

```html
<ai-not-found query="unicorn species"></ai-not-found>
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| query | `string` | `''` | What AI thought existed |
| suggestion | `string` | `''` | Alternative suggestion |

## Events

None
