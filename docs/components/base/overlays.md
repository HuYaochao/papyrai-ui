# Overlays

> Overlay components | 覆盖层组件

## Components

- [p-modal](./modal) - Modal dialog
- [p-popover](./popover) - Popover tooltip
- [p-tooltip](./tooltip) - Simple tooltip
- [p-toast](./toast) - Notification toast
- [p-slideover](./slideover) - Slide panel
- [p-context-menu](./context-menu) - Right-click menu
- [p-tour](./tour) - Product tour
- [p-watermark](./watermark) - Watermark overlay

## Usage

```javascript
// Toast example
import 'papyrai-ui/components/p-toast';
const toast = document.querySelector('p-toast');
toast.show({ message: 'Hello!', type: 'success' });
```
