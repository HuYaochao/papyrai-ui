# Layout

> Layout components | 布局组件

## Components

- [p-card](./card) - Card container
- [p-container](./container) - Layout container
- [p-skeleton](./skeleton) - Loading skeleton

## Usage

### Card

```html
<p-card>
  <div slot="header">Card Title</div>
  Card content here
</p-card>
```

### Container

```html
<p-container size="lg">
  Content
</p-container>
```

### Skeleton

```html
<p-skeleton width="200px" height="20px"></p-skeleton>
```
