# Getting Started

> Learn how to install and use papyrai-ui

## Installation

```bash
npm install papyrai-ui
```

## Quick Start

### Full Import

```javascript
import 'papyrai-ui';
```

```html
<ai-thinking></ai-thinking>
<p-button>Click me</p-button>
```

### Tree Shaking

Import individual components:

```javascript
import 'papyrai-ui/components/ai-thinking';
import 'papyrai-ui/components/p-button';
```

## CDN Usage

```html
<script src="https://unpkg.com/papyrai-ui"></script>
```
