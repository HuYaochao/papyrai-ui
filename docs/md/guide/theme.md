# Theme

> Customize papyrai-ui appearance

## Light Theme

The default theme. Uses paper textures, light colors, and serif fonts.

## Dark Theme

Switch to dark theme:

```javascript
document.documentElement.setAttribute('data-theme', 'dark');
```

## CSS Variables

Override any CSS variable to customize:

```css
:root {
  --paper-white: #ffffff;
  --paper-cream: #f7f3e8;
  --ai-thinking: #8b5cf6;
  --font-serif: 'Your Font', serif;
}
```
